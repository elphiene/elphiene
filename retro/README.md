# Retro Version - Cherry's Labs

This is the retro/Y2K/old web aesthetic version of Cherry's Labs.

## Viewing Locally

Simply open `index.html` in your web browser.

## Pages

- `index.html` - Homepage with intro to cherry and el
- `projects.html` - Project showcase
- `programs.html` - Software/development work
- `store.html` - E-commerce for 3D prints and gadgets
- `wiki.html` - Documentation and knowledge base

## Design Theme

**Y2K/Retro Web Aesthetic**
- Colors: Black, Pink (#ec4899), Purple (#9354d3)
- Sharp corners (no border radius)
- Pink glow effects
- Courier New font
- Web badges, blinkies, and stamps
- Marquee scrolling animations

All styling is in `style.css` using CSS custom properties for easy theming.

## Images

Organized in `/images` subdirectories:
- `art/` - ASCII and digital art
- `backgrounds/` - Background patterns
- `badges/` - Web badges (88x31)
- `blinkies/` - Animated blinkies
- `headers/` - Header graphics
- `icons/` - Icons and favicon
- `logos/` - Site logos
- `misc/` - Miscellaneous graphics
- `photos/` - Photography
- `stamps/` - Web stamps
- `stickers/` - Sticker graphics
